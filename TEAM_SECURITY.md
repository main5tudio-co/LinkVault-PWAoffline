## 📌 Overview

This document outlines the security measures and responsibilities for team members accessing LinkVault during alpha/beta testing phase.

## 🔒 Access Levels

| Resource | Access | Restriction |
|----------|--------|-------------|
| **GitHub Repo** | Private (invite only) | Team push access |
| **Netlify App** | Password protected | Invite only |
| **APK File** | Controlled distribution | Approved testers only |
| **Source Code** | PRIVATE | Not visible in APK |

## 👥 Team Member Responsibilities

By accessing this project, team members agree to:

### ✅ DO:
- ✅ Test features and report bugs
- ✅ Push code to main branch with clear commit messages
- ✅ Use password-protected Netlify URL only
- ✅ Keep Netlify password confidential
- ✅ Document work in commit history

### ❌ DO NOT:
- ❌ Share GitHub access with external parties
- ❌ Distribute Netlify URL/password publicly
- ❌ Fork repo to personal account without permission
- ❌ Download/distribute APK publicly
- ❌ Modify code without team discussion
- ❌ Copy source code for external use

